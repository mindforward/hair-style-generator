# Hair Style Generator

AI-powered hairstyle portrait generator. Take a selfie, choose a style, and generate 9 different hairstyle portraits in a 3x3 grid.

Built with **Gemini 3.1 Flash Image** (Google Interactions API).

## Features

1. **Pick Style** – Drag & drop tags (Culture / Aesthetic / Decade / Texture)
2. **Take Photo** – Camera selfie capture
3. **Generate** – Gemini 3.1 Flash Image transforms your photo into 9 styled headshots
4. **Print** – 4R photo frame with optional border upload

## Usage

1. Open `index.html` in a browser (or local server: `python3 -m http.server 8080`)
2. Click ⚙️ to set your Google Gemini API Key
3. Choose style tags
4. Take a selfie
5. Click "Generate" to create 9 styled headshots
6. Print or download the result
